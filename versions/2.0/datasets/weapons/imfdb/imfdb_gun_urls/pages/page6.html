If you've seen a gun in film, television, or anime that isn't listed here please feel free to create a page for it. To keep pages to a minimum and full of content we've consolidated all variants of a gun to a single page. If you're creating a new page for a gun make sure to add [[Category:Gun]] to the page so that it will be listed here.
